# ランアクションゲーム（仮）

## 概要
本作は、ワンボタン操作でキャラクターをジャンプさせ、  
迫ってくる敵を避けながらスコアを伸ばす横スクロール型アクションゲームです。

プレイヤーはジャンプのタイミングのみを操作し、  
敵の配置や速度、アイテムの出現によって変化する状況に対応していきます。

シンプルな操作性でありながら、  
スコアに応じて難易度が上昇する設計とし、  
繰り返し遊びたくなるゲーム体験を目指しました。

## 遊び方
1. ゲーム開始と同時に自動で前進します  
2. キーボードまたはマウス入力でジャンプします  
3. 敵に当たらないように回避しながら進みます   
4. アイテムを取得すると追加スコアを獲得できます  
5. 敵に接触するとゲームオーバーになります  

## 操作方法
- キーボード：任意のキー  
- マウス：クリック  
※ 押している時間によってジャンプ挙動が変化します

## 対応プラットフォーム
- PC（Webブラウザ）

## 使用技術
- JavaScript  
- HTML  

## プログラム面での工夫
### クラス設計
- Sprite クラスを基底クラスとして定義
- Player / Enemy / Item / Particle / Star を継承で実装
- 共通処理を親クラスにまとめ、拡張しやすい構成を意識

### シーン管理
- GameMain / GameOver の2シーンを定義
- シーンごとに更新処理・描画処理を分離
- 状態遷移を明確にし、処理の見通しを良くしています

### 難易度調整
- スコアに応じて敵の移動速度と出現頻度が上昇
- プレイが進むほど緊張感が高まる設計
- 敵の出現位置にランダム性を持たせ、単調さを軽減

### 演出・視覚表現
- 星・月・城を用いた背景演出で奥行きを表現
- ゲームオーバー時にパーティクルを大量生成し、派手な演出  

## ゲームバランス
- 敵回避だけでなく、アイテム取得によるリスクとリターンを意識した設計
- ジャンプのタイミング次第で  安全性とスコア効率が変化するよう調整しています

## まとめ
本作では、クラス設計・シーン管理・難易度スケーリングを意識し、  
シンプルなルールの中に奥行きのあるゲーム体験を組み込みました。
また、演出とロジックを分離して実装することで、  
調整や拡張がしやすい構造を学ぶことができました。
